PROJECT OVERVIEW
================================================================================

ENVIRR Institute is a professional educational coaching and tutoring website for
students in Classes VI through XII. The website serves as an online platform to
showcase the institute's services, faculty, courses, and provide an enquiry
system for prospective students to book demo classes and get in touch.

TAGLINE: "Shaping Futures, One Student at a Time"

WEBSITE URL: http://localhost:5174/ (Development)

ENVIRONMENT CONFIGURATION
================================================================================

PROJECT FILES:
- .env.local: Contains EmailJS credentials (not in git)
- .gitignore: Excludes sensitive files and node_modules
- tailwind.config.js: Tailwind CSS configuration
- postcss.config.js: PostCSS with @tailwindcss/postcss plugin
- vite.config.js: Vite build configuration
- package.json: Dependencies and npm scripts

KEY DEPENDENCIES INSTALLED:
- npm install: Installs all packages from package.json
- npm run dev: Starts Vite development server (default port 5173)
- npm run build: Creates production build
- npm run lint: Runs ESLint code quality checks
- npm run preview: Preview production build locally


PRODUCTION DEPLOYMENT CONSIDERATIONS
================================================================================

1. ENVIRONMENT VARIABLES
   - Store EmailJS credentials in .env files
   - Never commit sensitive data to repository
   - Use CI/CD tools to inject secrets during deployment

2. EMAIL SERVICE
   - Ensure Gmail account is authorized in EmailJS
   - Gmail needs "Less secure apps" access or OAuth2 setup
   - Monitor sent emails in EmailJS dashboard

3. FORM DATA
   - Currently sends to configured email address only
   - No database storage of enquiries (optional enhancement)
   - Could add backend database for enquiry history

4. SECURITY
   - Input validation on form fields
   - EmailJS handles CORS automatically
   - Consider CAPTCHA for production to prevent spam

5. PERFORMANCE
   - Tailwind CSS optimizes unused classes
   - Vite provides fast hot module replacement
   - Images should be optimized before deployment

TROUBLESHOOTING NOTES
================================================================================

CSS NOT APPLYING:
- Issue: Tailwind v4 syntax requires @import "tailwindcss"
- Solution: Created postcss.config.js with @tailwindcss/postcss plugin
- Resolved: Content path must be lowercase "content" not "Content"

EMAIL NOT SENDING:
- Issue: Gmail authentication scopes insufficient
- Solution: Reauthorize Gmail in EmailJS with full permissions
- Check: Template ID updated to template_gbs0ujw

PORT CONFLICTS:
- If 5173 is in use, Vite automatically uses 5174
- Multiple instances of dev server can run on different ports
